This plugin allows you to add a high-conversion mobile CTA with two buttons, as well as inject any code you need into your theme headers (fb pixel, analytics, etc) and enable Font Awesome icons.

== Description ==

The core feature of the plugin is the high-conversion mobile CTA that is guaranteed to increase your click-to-call and contact rates since it is well designed and customizable. The plugin also allows you to inject any code you need into your headers (like fb pixel and Google Analytics) and allows you to customize the colors and enable icons on the CTA.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Access the ‘Ascend Marketing’ menu in your dashboard to use the plugin.

== Frequently Asked Questions ==

= Can I change the colors of the CTA? =

Yes, you can choose to change the background color, text color, and the background of the buttons.

= Can I determine what width the CTA appears at? =

Yes, you can enter a value in pixels that will be the width below which the CTA appears
